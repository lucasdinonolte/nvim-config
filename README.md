# Neovim Config

My Neovim config based on [NvChad](https://nvchad.com/)

## Install

```bash
# Remove old nvim config (if needed)
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim

# Clone the config
git clone git@github.com:lucasdinonolte/nvim-config.git ~/.config/nvim

# Open Neovim
nvim
```

After lazy.nvim finished installing all plugins run `:MasonInstallAll`
